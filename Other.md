# 🚀 Free LLM APIs & Resources

A curated collection of free LLM APIs, proxies, gateways, and awesome lists for developers looking to integrate AI without high costs.

## 📚 Curated Lists & Collections
Comprehensive repositories and sites that track active free providers and API keys.

| Project | Description | Link |
| :--- | :--- | :--- |
| **Awesome Free LLM APIs** | A massive list of free LLM APIs for developers. | [GitHub](https://github.com/amardeeplakshkar/awesome-free-llm-apis) |
| **Free-LLM-Collection** | A collection of various free LLM resources. | [GitHub](https://github.com/for-the-zero/Free-LLM-Collection) |
| **Awesome Free AI API** | Another great curated list of free AI services. | [GitHub](https://github.com/YoannDev90/awesome-free-ai-api) |
| **Free LLM Providers** | A web-based dashboard tracking LLM providers. | [Site](https://free-llm-apis.pages.dev/providers/) |
| **Free AI APIs** | A simple directory of open AI endpoints. | [GitHub](https://github.com/OuterSpacee/free-ai-apis) |
| **Free For Life** | A broad list of free services for developers (includes AI). | [GitHub](https://github.com/wdhdev/free-for-life) |
| **Free AI Tools** | Tools and resources for accessing AI for free. | [GitHub](https://github.com/ShaikhWarsi/free-ai-tools) |
| **Cool AI Stuff** | A miscellaneous collection of interesting AI projects. | [GitHub](https://github.com/zukixa/cool-ai-stuff) |
| **Free LLM API Keys** | A repository dedicated to sharing accessible API keys. | [GitHub](https://github.com/alistaitsacle/free-llm-api-keys) |

## 🛠 Proxies, Gateways & Aggregators
Tools designed to bridge, proxy, or consolidate multiple AI services into a single interface.

| Project | Description | Link |
| :--- | :--- | :--- |
| **GPTOSS Proxy** | A proxy service for various GPT-based models. | [GitHub](https://github.com/junioralive/gptoss-proxy) |
| **Kiro Gateway** | A high-performance gateway for LLM traffic. | [GitHub](https://github.com/jwadow/kiro-gateway) |
| **Gemini Proxy** | Specifically designed to proxy Google's Gemini API. | [GitHub](https://github.com/KashifKhn/gemini-proxy) |
| **AI Fiesta Unlimited** | An aggregator for unlimited AI API access points. | [GitHub](https://github.com/cu-sanjay/ai-fiesta-unlimited) |
| **Sixfinger API** | A specialized API hub for AI model integration. | [GitHub](https://github.com/sixfingerdev/sixfinger-api) |
| **Blacklisted AI Proxy** | A proxy focused on bypassing certain restrictions. | [GitHub](https://github.com/crazyrob425/BlacklistedAIProxy) |
| **G4F Provider Pulse** | Monitoring tool for GPT4Free (G4F) provider status. | [GitHub](https://github.com/Stingerva/g4f-provider-pulse) |
